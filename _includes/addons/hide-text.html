Hide-text

View Source

Hide-text is an image replacement mixin. It is based off the HTML5-Boilerplate image-replacement.

This image replacement technique is an alternative to the text-indent technique, which has performance and other benefits.

A height declaration is required for this to function.

{% highlight scss %} .element { @include hide-text; background-image: url(logo.png); } {% endhighlight %}

CSS Output

{% highlight scss %} .element { } {% endhighlight %}

You can optionally pass a height value:

{% highlight scss %} .element { @include hide-text(2em); } {% endhighlight %}
